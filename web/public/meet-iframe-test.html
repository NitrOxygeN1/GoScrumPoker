Same-origin caveat. Because this page is on goscrumpoker.onrender.com, the iframe shares cookies and storage with the top window — that's not what real Meet does (Meet embeds you from meet.google.com as a third-party context). This page is great for layout and embedded-mode UI checks; for cookie/storage partitioning behavior, use /meet-debug inside Meet.
